# MarkdownFlow 语法规范（统一版）

1. 变量：
- 用 `{{var_name}}` 引用变量；
- 变量名不可含空格；
- 未赋值变量默认 `"UNKNOWN"`。

2. 交互：
- 单选：`?[%{{var}} 选项A | 选项B | 选项C]`
- 多选：`?[%{{var}} 选项A || 选项B || 选项C]`
- 输入：`?[%{{var}} ... 请输入]`
- 按钮+输入：`?[%{{var}} 选项A | 选项B | ...其他，请填写]`

3. 分镜：
- 用 `---` 分隔模块；
- 每个模块只完成一个明确目标。

4. 确定性输出：
- 单行固定文本：`===固定文本===`
- 多行固定文本围栏：

```md
!===
第1行
第2行
!===
```

5. 普通内容写作原则：
- 普通内容是“给AI的创作指令”，不是直接给读者的成文；
- 禁止直接输出整篇文章正文，应指导AI生成正文。
